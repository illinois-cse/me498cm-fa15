Natural Conection in a Square Cavity¶

Introduction¶

Natural convection flows are encountered in various engineering applications. In this lab, you will solve a steady state natural convection problem in a 2D square cavity. The flow is induced by thermal differential between the left and right walls that are held at constant hot and cold temperatures, respectively. The top and bottom walls are insulated (i.e. zero heat flux). The fluid near the left wall ascends because it is hotter than the ambient, while the fluid near the right wall descends because it is cooler, hence generating a clockwise vertex. Geometry and Mesh¶

  • Download the Fluid Flow (Fluent) Component System for this tutorial.
  • Extract the files to a new folder and name it as you wish.
  • Open the ANSYS Workbench file and select the Mesh Component System.
  • After importing the downloaded mesh, transfer data to new Fluent. You can do this by right clicking the Mesh cell and then selecting Transfer Data to New and then select Fluent.

Setup¶

  • You will solve a steady state laminar natural convection of Rayleigh number 10,000.
    • Enter FLUENT with Double Precision.
    • Change temperature unit to $Celsius$.
    • Go to Materials menu and change the density option to $boussinesq$ and type in the numbers as shown below.

The density is the ρ$_0$ in the Boussinesq approximation.

  • Next, go to Boundary Conditions menu.

    • Set the boundary conditions for the four walls.
    • The top and bottom walls have zero heat flux.
    • In order to set these, click on Edit after selecting bottomwall. Select the Thermal tab and select the Heat Flux option. Enter $0$ for Heat Flux and Heat Generation Rate.

    • Set the same boundary condition for the top wall.

    • The left and right walls have constant temperature at $1$ and $0$, respectively. (This is arbitrary and will scale.)

    • In order to set these, click on Edit after selecting leftwall. Select the Thermal tab and then select Temperature. Enter $1$ for Temperature.

    • Set the boundary condition of the right wall accordingly

  • Now open the Operating Conditions menu in the Boundary Conditions section itself.

    • Check the Gravity box and enter $-1$ in $Y$ direction.
    • Enter $0.5$ for the Operating Temperature that corresponds to $T_0$ in the Boussinesq approximation.
    • You do not need to specify operating density when using the Boussinesq approximation.

    • Leave the Specified Operating Density unchecked.

  • Initialize and start the calculation. If your setup is correct, it should converge in $500$ iterations.

Post-processing¶

  • Plot the contours of temperature, pressure, and velocity.

  • Plot the $Y$-velocity along the horizontal centerline and $X$-velocity along the vertical centerline.

Do the results make sense? Compare your results to the benchmark solution (de Vahl Davis 1983). The materials are different. So the magnitude will not match.
